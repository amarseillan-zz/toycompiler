PROGRAM -> PREPROCESSOR FUNCTION_DEFS FUNCTIONS
PREPROCESSOR -> PREPROCESSOR PREPROCESSOR_STATEMENT
FUNCTIONS -> FUNCTIONS TYPE ID ( FORMAL_PARAMETERS_LIST ) { P } | lambda
FORMAL_PARAMETER_LIST -> FORMAL_PARAMETER | FORMAL_PARAMETER , FORMAL_PARAMETER_LIST | lambda
FORMAL_PARAMETER -> TYPE ID
P -> P I ; | P D ; | P CONTROL_SEQ | lambda
D -> FORMAL_PARAMETER | TYPE ID EQUALS ASSIGN | TYPE ID EQUALS CONST | RETURN ASSIGN
I -> ID EQUALS ASSIGN | ID EQUALS CONST | FUNCTION_CALL
FUNCTION_CALL -> ID( PARAM_LIST ) | IO_CALL
PARAM_LIST -> ASSIGN | CONST | CONST , PARAM_LIST | ASSIGN , PARAM_LIST
ASSIGN -> ID | ( ASSIGN ) | ASSIGN + ASSIGN | ASSIGN - ASSIGN | ASSIGN * ASSIGN | ASSIGN / ASSIGN | ASSIGN + CONST | ASSIGN - CONST | ASSIGN * CONST | ASSIGN / CONST | CONST + ASSIGN | CONST - ASSIGN | CONST * ASSIGN | CONST / ASSIGN | CONST + CONST | CONST - CONST | CONST * CONST | CONST / CONST | FUNCTION_CALL
CONTROL_SEQ -> IF ( CONDITION ) { P } | WHILE ( CONDITION ) { P }
CONDITION -> ASSIGN COND_OP ASSIGN | CONST COND_OP ASSIGN | ASSIGN COND_OP CONST | CONST COND_OP CONST
COND_OP -> == | > | < | => | =<
